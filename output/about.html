My name is Yannis and I live and work in Belgium as a cloud developer. Sometimes data, sometimes infra, sometimes just coding. Whatever happens to need fixed.

This is a simple blog built with a minimal vibe-coded Python-based markdown blog generator.

It supports:

  • Markdown formatting
  • Syntax highlighted code blocks
  • Copy buttons on code snippets
  • Customizable themes via config.yaml

I tried using static site generators but gave up because the complexity was just not worth the effort. After all, all I needed was just some way to publish basic texts. Keep it simple!

You can find the source code in my github page.
